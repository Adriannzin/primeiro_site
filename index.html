Hospedagem Github

Para criar um site no Github.com devemos ter uma conta válida e um repositório com um arquivo index.html

Hospedando um site

  • Criar um repositório
  • Publicar o repositório utilizando o publish
  • Abrir o repositório no VSCode
  • Criar o index.html
  • Criar a versão usando Commit
  • Enviar para o repositório usando push
  • Entrar no github.com, acessar o repositório -> Settings -> Pages -> Escolher a branch do source
  • Clicar na URL gerada

    • Branches

    São ramificações do seu projeto, o projeto principal em geral fica na branch main. A branch main deve ter a versão final e funcional do seu projeto. O projeto pode ter branches secundárias.
